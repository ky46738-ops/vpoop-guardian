💻 Python 版指令碼

不想用瀏覽器？直接跑 Python：

# v💩 Guardian — 計算 SHA256
import hashlib

def vpoop_hash(filepath):
    sha256 = hashlib.sha256()
    with open(filepath, "rb") as f:
        for chunk in iter(lambda: f.read(65536), b""):
            sha256.update(chunk)
    digest = sha256.hexdigest()
    print(f"你的💩指紋：{digest}")
    return digest

# 使用方式
my_hash = vpoop_hash("contract_v1.pdf")

# 比對（驗證對方有沒有改）
their_hash = input("貼上他聲稱的 SHA256：").strip()
if my_hash == their_hash:
    print("✅ A_EQUALS_A = True  → 這💩確實是你拉的")
else:
    print("💩 A_EQUALS_A = False → 有人說謊。他💥謝謝。")

🧠 為什麼 SHA256 能讓說謊者💥謝謝？

🔢 唯一性

相同的輸入永遠產生相同的輸出。任何一個字元被改動，SHA256 就完全不同。改了逗號？不同。改了空格？不同。改了日期？不同。

🔒 不可逆性

給你 SHA256，你無法還原成原始檔案。它是指紋，不是鑰匙。存了它，你就有了所有權的時間戳，對方無法偽造。

⚡ 即時性

本工具完全在瀏覽器本地運算。不需要網路、不需要帳號、不上傳任何資料。你的機密合約永遠不離開你的電腦。

🌐 通用性

SHA256 是 ISO 標準。法院、銀行、區塊鏈、數位簽章都用這個。你的證據格式，全球通用。

❓ 常見問題

Q：這有法律效力嗎？

SHA256 + 時間戳在台灣《電子簽章法》框架下具有參考效力，但不等同法院判決。它是你的第一道舉證材料，不是最後的終極武器。搭配律師才是正解。

Q：我的檔案會被上傳嗎？

絕對不會。本頁所有計算使用瀏覽器原生 Web Crypto API，完全本地運算。沒有伺服器，沒有 cookie，沒有追蹤。

Q：對方也能算出相同 SHA256 偽造嗎？

理論上找到碰撞的機率是 1/2^256，比宇宙裡的原子數量還少。實務上：不可能。SHA256 目前尚無已知碰撞攻擊。

Q：開源版跟商業版差在哪？

開源版讓你自己驗，適合個人和開發者。商業版幫你自動存證、上鏈、寄函，適合企業。兩個的核心邏輯完全相同。
