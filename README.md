## 這是什麼？

v💩 Guardian 是一套**數位著作權驗證系統**，分三層：

| 層級 | 檔案 | 用途 | 費用 |
|---|---|---|---|
| 🛡️ 防禦型（開源） | `vpoop_guardian.py` | SHA256 完整性驗證 | $0（傳教） |
| 🕊️ 仁慈版（三抽） | `vpoop_guardian_mercy.py` | 三振出局，給三次機會 | $5,000/年 |
| ☢️ 語意核彈（顧問） | `vpoop_vector_semantic.py` | 語意向量比對，連改字都抓得到 | $20,000/案 |

---

## 快速開始

```bash
pip install sentence-transformers numpy
python vpoop_guardian.py
```

---

## 核心原則

1. **SHA256** 防「一模一樣的抄」
2. **語意向量（餘弦相似度 > 0.88）** 防「改幾個字的抄」
3. **三抽定律** 給三次機會，第三次 💥謝謝
